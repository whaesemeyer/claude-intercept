Welcome to Claude Intercept

A lightweight MITM proxy that captures HTTP/S traffic from your Mac, iPhone, or any device — then lets you feed it directly to Claude for analysis.

🔍
Capture all traffic
Intercept HTTP and HTTPS from browsers, apps, CLI tools, and mobile devices in real time.
🔑
Extract auth & tokens
Automatically surface API keys, Bearer tokens, cookies, and JWTs from captured traffic.
📄
Document undiscovered APIs
Export captured endpoints as structured Markdown — paste straight into Claude for analysis.
📱
Intercept phones too
Works with iOS, Android, or any device on the same network. QR-code cert install included.

Install the CA Certificate

To read HTTPS traffic, your device needs to trust Claude Intercept's certificate authority. This is a one-time setup that takes about 30 seconds.

1
Download the certificate
Click the button below to save claude-intercept-ca.crt to your Downloads folder.
⬇ Download CA Certificate
2
Trust it on macOS
Run this command in Terminal (you'll be prompted for your password):
sudo security add-trusted-cert -d -r trustRoot \
  -k /Library/Keychains/System.keychain \
  ~/Downloads/claude-intercept-ca.crt
Your privacy is protected. The CA private key is stored only on this machine at ~/claude_intercept/certs/ca.key and is never transmitted anywhere.
🌐

Enable the Proxy

Claude Intercept is a standard HTTP/S proxy running on port 8080. You can route traffic from this Mac or any device.

A
This Mac — one click
Go to Setup & Certs → This Mac and click Enable for This Mac. All browser and app traffic will be intercepted instantly. Disable with one click when done.
B
iPhone / iPad
Go to Setup & Certs → iOS / iPadOS for a step-by-step guide with a QR code to install the cert on your phone — no typing required.
C
Terminal / CLI tools
Most CLI tools respect env vars. Prefix any command or set for the session:
https_proxy=http://127.0.0.1:8080 curl -s https://api.example.com
🎉

You're all set

Once traffic starts flowing, here's what you can do:

1
Watch traffic in real time
The Traffic tab shows every request live — click any row to inspect headers and bodies.
2
Select captures and export to Claude
Check the boxes next to interesting requests, go to Export for Claude, choose a mode (API docs, auth extraction, full detail), and copy the output straight into Claude.
3
Use /intercept in Claude Code
In any Claude Code session, type /intercept to get AI-assisted capture management, analysis, and export — all from the terminal.
Remember: Disable the proxy when you're done (Setup & Certs → Disable). Leaving it on with the proxy stopped will break your network connection.
